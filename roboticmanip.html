Robotic Manipulation

Welcome to the project page for robotic manipulation ME 449 at Northwestern University. Below you will find a description of the project, a video of the robot in action, and a link to the code.

Robotic Manipulation

In this class we used a four wheeled robot with omni-wheels and a 6 degree of freedom
arm to pick a block up and move it to a specified location. The robot was controlled using
inverse kinematics to calculate the path of the robot and the arm. We used the modern
robotics library in Python to calculate the inverse kinematics.The robot was then simulated
using CopeliaSim. The code for the project as well as additional information can be found
on my github page.
